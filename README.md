GETTING STARTED:
---------------
1. Launch the application
2. Choose your favorite theme
3. Navigate to any folder
4. Start organizing!

QUICK TIPS:
-----------
- Right-click files for quick actions
- Use Ctrl+F to search
- Press F2 to rename
- Drag files to move them
- Add folders to favorites with the ⭐ button

KEYBOARD SHORTCUTS:
------------------
Ctrl+N          - New File
Ctrl+Shift+N    - New Folder
Ctrl+C/X/V      - Copy/Cut/Paste
Delete          - Delete Files
F2              - Rename
F5              - Refresh
Ctrl+F          - Search
Ctrl+A          - Select All
